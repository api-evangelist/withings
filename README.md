# Withings (withings)

Withings is a connected health device platform providing a REST API for accessing data from smart scales, blood pressure monitors, sleep trackers, thermometers, and fitness watches via OAuth2. The API enables partners to integrate Withings health biomarkers including body composition, cardiovascular data, sleep metrics, and activity tracking into wellness apps, remote patient monitoring platforms, and clinical research tools.
